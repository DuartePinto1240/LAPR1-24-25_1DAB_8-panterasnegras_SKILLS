Sobre o Projeto

A aplicação proposta surge com o objetivo de facilitar o processo de preenchimento e compreensão do IRS, fornecendo uma interface amigável e acessível para os usuários. Ela busca simplificar o entendimento das obrigações fiscais, ajudar na correta inserção de dados e garantir que o contribuinte possa maximizar seus benefícios fiscais. Além disso, a aplicação oferecerá uma explicação clara dos conceitos fiscais envolvidos, capacitando os utilizadores a tomarem decisões mais informadas.

Objetivo do Projeto

O projeto tem como metodologia a criação de uma plataforma digital que incorpora uma base de dados atualizada sobre a legislação fiscal em vigor, algoritmos de cálculo que verificam os dados inseridos e sugerem otimizações fiscais, além de uma interface que traduz as complexidades técnicas do IRS para uma linguagem mais acessível. A aplicação será aplicável de maneira geral a todos os contribuintes portugueses, mas poderá ser adaptada para cenários específicos de diferentes perfis fiscais, como trabalhadores por conta própria ou dependente, entre outros.
